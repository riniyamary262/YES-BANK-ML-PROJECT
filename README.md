# YES-BANK-
ML PROJECT - INTERNSHIP MINOR PROJECT

This machine learning project successfully developed three regression models - Linear Regression, Random Forest, and Decision Tree - to predict Yes Bank's monthly closing stock prices using historical price data from 2005 to 2020. After comprehensive analysis and hyperparameter tuning, the Tuned Random Forest Regressor emerged as the best performing model.

The final model achieved impressive results with an R² score of 0.941, meaning it can explain 94.1% of the price variations. The average prediction error was just ₹10.89 (RMSE), and the mean absolute percentage error was only 6.8%, demonstrating reliable and accurate price forecasts. The model also showed consistent performance across cross-validation folds, confirming its stability and generalizability.

The analysis revealed that previous month's closing price was the most important feature (22.4% importance), followed by current month's opening price and moving averages. This confirms that stock prices follow momentum patterns, and historical prices are strong predictors of future prices. Feature importance insights help investors focus on the most influential factors when making investment decisions.

For investors and traders, this model provides significant business value. It enables reliable price predictions for better investment timing, risk management through volatility assessment, and data-driven decision making. The model has been saved and is ready for deployment in production environments, making it a practical tool for real-world use.

This project successfully demonstrates that machine learning can effectively capture patterns in stock price data and provide actionable insights for investment strategies.
